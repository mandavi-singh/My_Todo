About MyTodo

MyTodo is a simple and elegant task management app built using Flask, SQLite, and Bootstrap. It helps you keep track of your daily tasks, update them when needed, and delete tasks once they’re done ✅.

Features include:

  • Add and manage your todos easily
  • Edit tasks anytime
  • Delete completed tasks

This project is created as part of learning Flask and SQLAlchemy. It’s a great way to practice building full-stack web applications!
